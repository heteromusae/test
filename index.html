📌 프리셋 관리 및 자동 저장

입력한 로그와 설정은 새로고침을 해도 로컬 스토리지에 자동 저장됩니다. 마음에 드는 폰트, 여백, 색상 설정 조합은 [새 프리셋으로 저장]을 눌러 언제든 다시 불러올 수 있습니다.

⚙️ 텍스트 자동 변환

일괄 변환: 별표로 감싸지지 않은 일반 텍스트 문단에 지정한 따옴표를 입히고, 지문에 붙은 별표를 모두 삭제합니다.
웹 서식 정리: 외부에서 복사한 HTML 코드를 정리하여 불필요한 태그와 스타일은 지우고 텍스트만 깔끔하게 보존합니다.

✨ 상단 툴바 기능

실행 취소 / 되돌리기: 화살표 버튼을 통해 작업 내역을 되돌릴 수 있습니다.
M↓ 마크다운: *내용* 형식으로 작성시 기울임꼴과 함께 투명도 낮춤. 그 외에도 모든 마크다운 양식이 적용됩니다.

📸 이미지 저장 및 자동 자르기

이미지 비율을 선택한 후 [발췌본 만들기]를 클릭하세요. [마지막 페이지 자동 자르기]가 켜져 있으면, 글이 끝나는 마지막 장에서 남는 잉여 공간을 잘라내 최종 출력본이 완성됩니다.
